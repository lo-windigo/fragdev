Projects

This is a small collection of some random projects I've done over the years. They're all open source, so feel free to use them. I hope to add more as I dig them out of archives.

Getreel

A web service for people with data caps. This service allows you to add videos to be downloaded at a schedule time, and played as files or via servers like MuttonChop.

I created the responsive GUI for Getreel.

jQuery DynMap Chat Widget

I play a lot of Minecraft, and help run a server or two. So that we could display the server chat on our website, I whipped up a little chunk of jQuery that would pull the chat text from the DynMap plugin.

BarCamp Badge Creator

I was lucky enough to attend the first and second OggCamp un-conferences, and was able to meet quite a few people I had gotten to know online. There was just one problem... how do you identify someone you've known only via forum avatars?

A quick coding sprint later, I had a small PHP script that would generate you a "Hello, My Name Is" image that you could print out and stick on your shirt/laptop/etc.

Florrie: Comic Management System

This is a refactored, incomplete version of a PHP4 Comic management system I created for some friends that run the Val'Cielle web comic.

I've put this project on hold mostly, due to the comic also being put on hold. I'm planning on finishing this at some point, possibly in Django.

Hacker Public Radio

As someone who enjoys the akward, knobly bits of technology and the stranger, more obscure aspects of DIY, I'm an avid listener and occasional contributer to the Hacker Public Radio network. Feel free to check out some of my contributions, and more importantly, make some of your own. Hacker Public Radio is always looking for new hosts!

Silly MPD Script

While on the quest for the perfect media player (still unfinished, I'm afraid) I created a small bash script that would control MPD. I then bound these functions to the audio keys in my OpenBox config, so that I could switch songs, pause playback, and more with a single keystroke.
