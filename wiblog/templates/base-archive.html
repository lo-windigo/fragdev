History

This blog has gone through a couple different permutations. Originally a static blog that I wrote in plain HTML to learn the dvorak keyboard layout (still haven't, by the by), it was then transitioned to a LifeType blog.

After it was clear that LifeType wasn't undergoing too much active development, and the comment spam became a little hard to handle, I disabled the lifetype blog and threw up a static PHP site sans-blog.

The latest iteration was created in Django, mostly as an exercise to help me learn Django.
